# Changelog

## v1.0.0 (2026-02-16)

Initial release.

### Features

- 47都道府県 + 主要都市200以上の辞書による地域キーワード自動分類
- 9種類の分析シート自動生成
  - サマリー（全体概要・パフォーマンス比較・カバレッジ）
  - 都道府県別分析（KW数・クリック・表示・CTR・順位・Top3/Top10）
  - 地方別分析（7地方ブロック集計）
  - 地域KW一覧（マッチ根拠付き全件リスト）
  - チャンスKW（高表示・低順位のKW自動抽出）
  - 低CTR改善KW（順位帯別CTRベンチマーク比較）
  - 未カバー地域（47都道府県のカバー状況・色分け）
  - 順位帯別分布（都道府県×順位帯クロス集計）
  - 地域×サービス マトリクス（設定シート連動）
- 日本語/英語ヘッダー両対応
- 1,000件以上のクエリデータ対応
- 表示回数加重平均順位の算出
- カスタムメニューからのワンクリック実行
